# 🎤 Python Lyrics Printer

A simple Python CLI script that displays song lyrics in the terminal with a **typewriter animation effect** and **colorful text** using ANSI escape codes.

## ✨ Features

- 🎨 Colored terminal output (Pink, Blue, Cyan, Green, Yellow) using ANSI escape codes
- ⌨️ Typewriter-style character-by-character printing
- ⏱️ Custom delay timing between lines for a smooth, dramatic reveal
- 🖥️ Cross-platform compatible (with UTF-8 encoding handling for Windows)

## 🛠️ Tech Stack

- **Language:** Python 3
- **Libraries used:** `sys`, `time`, `os`



## 📚 What I Learned

- How ANSI escape codes work to color terminal text
- Building a typewriter animation effect using `time.sleep()`
- Debugging encoding issues (`cp1252` vs `UTF-8`) on Windows terminals
- Writing clean, readable Python scripts for CLI tools
